04What has been built

  • Decision API: single, bulk (1-500), and stream (1-5000) evaluation for raw and intake request models.
  • Policy-as-code engine: unified Rego policy set backing runtime checks and compliance reporting.
  • Frontend audit experience: public HTTPS upload UI with API-key-gated submissions for self-service firewall policy audit.
  • Multi-vendor import pipeline: Fortinet XLSX, Palo Alto CSV, and Juniper SRX JSON/XML uploads normalised into one audit path.
  • Compliance scope controls: always-on Clarisys NFR baseline with selectable ISO 27001, CIS v8.1, and PCI-DSS standards in the UI.
  • Audit and governance: HTML report generation, cleaned CSV/JSON artifact export, decision history, lifecycle updates, evidence export, and drift recheck endpoints.
  • Operational controls: health endpoints, SLO/alert metrics, cache stats, and structured notifications.
  • Resilience hardening: atomic state writes, startup state drift detection/autocorrect, watchdog and backup jobs.
